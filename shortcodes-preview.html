Shortcode 01

Property Search Filter

Admin Panel

Gestion desde WordPress

El plugin agrega un panel lateral llamado HY Homes Syd para administrar propiedades, banners y carga masiva desde hojas de calculo.

action,type,id,slug,title,description,neighborhood,room_type,bedrooms,bathrooms,street,address,price,availability_date,availability,price_suffix,status,move_in,detail_url,featured_image_url,gallery_media,map_embed_url,image_url,button_url

Properties

Crear, editar, eliminar y publicar propiedades con localidad, precio, galeria, mapa y datos de filtro. WhatsApp se configura una sola vez con tres agentes globales.

Location Banners

Crear, editar y eliminar banners del carrusel, asignando cada banner a una o mas localidades.

Excel / Google Sheets

Importar CSV, XLSX o una hoja de Google publicada como CSV. Para borrar, usar action=delete con id o slug.
